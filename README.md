# Video Game Sales Analysis

## Project Overview
This project analyzes the Video Game Sales dataset using Python. It focuses on data cleaning, regional sales comparison, yearly sales trends, publisher performance, and market preferences across different regions through data visualization and business insights.

## Dataset
https://www.kaggle.com/datasets/gregorut/videogamesales

## Objectives
- Inspect the dataset
- Handle missing values
- Analyze global sales by Genre, Platform, and Publisher
- Compare regional sales shares
- Identify the most popular game genre in each region
- Analyze yearly global sales trends (1980–2016)
- Find the top 10 best-selling games
- Generate business insights

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Tasks
1. Loaded and inspected the dataset.
2. Removed missing values from the **Year** column and filled missing **Publisher** values with **"Unknown"**.
3. Calculated total global sales by Genre, Platform, and Publisher.
4. Compared regional sales shares (North America, Europe, and Japan).
5. Identified the most popular game genre in each region.
6. Analyzed yearly global sales trends from **1980 to 2016**.
7. Identified the top 10 best-selling video games along with their platform and genre.
8. Created visualizations to support the analysis.

## Visualizations
- Regional Sales by Genre (Stacked Bar Chart)
- Yearly Global Sales Trend (Line Chart)
- Top 10 Publishers by Global Sales (Bar Chart)

## Business Insights
- North America is the largest video game market, while Japan has different genre preferences.
- Action and Sports games dominate sales in North America and Europe, whereas Role-Playing games are more popular in Japan.
- Global video game sales increased steadily until the late 2000s before gradually declining.
- A small number of major publishers account for a significant portion of worldwide game sales.
